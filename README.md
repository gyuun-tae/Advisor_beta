# ADviser - AI 마케팅 솔루션

ADviser는 소상공인을 위한 AI 기반 마케팅 분석 및 진단 플랫폼입니다. 광고 성과를 자동으로 분석하고, 진단하며, 개선 방안을 제시합니다.

## 🚀 주요 기능

### 1. 광고 플랫폼 선택 및 관리
- **다중 플랫폼 지원**: 배달의민족, 쿠팡이츠, 요기요, 네이버, SNS/커뮤니티
- **하위 옵션 선택**: 각 플랫폼별 세부 광고 상품 선택
  - 배달의민족: 배민1플러스, 울트라콜, 우리가게클릭
  - 쿠팡이츠: 스마트요금제, 매출 연동
  - 요기요: 요기요 라이트, 추천광고
  - 네이버: 플레이스 상위 노출, 파워링크, 블로그 마케팅 대행
  - SNS/커뮤니티: 인스타그램, 당근마켓, 유튜브 마케팅 대행
- **다중 채널 동시 분석**: 여러 채널의 데이터를 한 번에 업로드하고 분석

### 2. OCR 기반 자동 데이터 추출
- **이미지 업로드**: 광고 관리자 페이지 스크린샷 업로드
- **자동 텍스트 인식**: Tesseract.js를 활용한 OCR 엔진
- **데이터 추출**: 노출수(Impression), 클릭수(Click), 전환수(Conversion) 자동 추출
- **다중 채널 지원**: 채널별로 이미지를 업로드하고 개별 분석
- **드래그 앤 드롭**: 파일 선택 또는 드래그 앤 드롭으로 업로드
- **실시간 진행률 표시**: OCR 처리 진행 상황 실시간 표시

### 3. 3단계 분석 플로우
#### 1단계: 분석 (Analysis)
- 광고 채널 선택
- OCR 데이터 입력
- 기본 성과 지표 확인

#### 2단계: 진단 (Diagnosis)
- ROAS(광고 투자 대비 수익) 분석
- 채널별 성과 비교
- 문제점 자동 진단

#### 3단계: 행동 (Action)
- 개선 방안 제시
- 구체적인 실행 계획 수립

### 4. 데이터 시각화
- **ROAS 차트**: 채널별 ROAS 비교 바 차트
  - 배민, 쿠팡, 네이버, 인스타그램 성과 비교
  - OCR 데이터 기반 동적 계산
  - 색상 코딩(좋음/보통/나쁨)
- **매출 차트**: 주간 매출 추이 바 차트
  - 요일별 매출 분석
  - OCR 데이터 기반 예측 매출 계산
  - 최고 매출일 하이라이트

### 5. 홈 대시보드
- **예상 매출 표시**: 이번 달 예상 매출 및 증감률
- **XP 시스템**: 레벨 및 경험치 표시
- **오늘의 퀘스트**: 일일 미션 목록
  - 메뉴 사진 업데이트 (+50 XP)
  - 리뷰 답글 달기 (+30 XP)
- **건강검진 시작**: AI 자동 진단 시작 버튼
- **연속 출석 표시**: 연속 출석 일수 표시

### 6. 성과 분석 화면
- **매출 탭**: 주간 매출 차트 및 상세 분석
- **ROAS 탭**: 채널별 ROAS 분석 및 비교
- **주간 매출 표시**: OCR 데이터 기반 계산된 주간 매출
- **증감률 표시**: 전주 대비 증감률

### 7. AI 리뷰 관리
- **리뷰 목록**: 답글 대기 중인 리뷰 표시
- **AI 답글 생성**: 리뷰 유형별 자동 답글 생성
  - 배달 지연 관련 리뷰: 정중한 사과, 보상 제안, 공감형 답글
  - 칭찬 리뷰: 감동형, 위트있는, 짧고 굵은 답글
- **답글 복사**: 생성된 답글을 클립보드에 복사

### 8. 성장 시스템
- **레벨 시스템**: 레벨 및 등급 표시 (GOLD, Platinum 등)
- **XP 획득**: 미션 완료 시 XP 획득
- **랭킹 표시**: 전체 사용자 대비 순위 (예: TOP 14%)
- **성장 미션**: 다양한 성장 미션 제공
  - 메뉴 사진 업데이트
  - 리뷰 답글 달기
  - 광고 데이터 업로드

### 9. 프로필 및 설정
- **프로필 정보**: 사장님 이름, 업종, 지역 표시
- **데이터 연동 설정**: 각 플랫폼별 데이터 연동 토글
  - 배달의민족
  - 쿠팡이츠
  - 요기요
  - 네이버 플레이스
- **멤버십 표시**: PRO 멤버십 상태 표시

### 10. 모달 및 팝업 시스템
- **평생 무료 모달**: 초기 진입 시 평생 무료 서비스 안내
- **광고 팝업**: 랜덤 광고 표시
  - 사장님 대출
  - 매출 관리 통장
  - 박카스 특가
- **보상 모달**: 미션 완료 시 XP 보상 표시
- **면책 조항 모달**: 서비스 이용 전 면책 조항 확인

### 11. 반응형 디자인
- **모바일 최적화**: 모바일 화면에 최적화된 UI/UX
- **데스크톱 지원**: 태블릿 및 데스크톱 화면 지원
- **터치 제스처**: 모바일 터치 제스처 지원

## 🚀 설치 및 실행

### 로컬 개발 환경

1. **저장소 클론**
```bash
git clone <repository-url>
cd FighTeam
```

2. **의존성 설치**
```bash
npm install
```

3. **개발 서버 실행**
```bash
npm run dev
```

4. **브라우저에서 접속**
- 메인 앱: `ADviser_web.html` 파일을 브라우저에서 직접 열기
- React 앱: `http://localhost:5173` (Vite 기본 포트)

### 프로덕션 빌드

```bash
npm run build
```

### Docker를 사용한 실행

1. **Docker 이미지 빌드**
```bash
docker build -t adviser-app .
```

2. **컨테이너 실행**
```bash
docker run -p 8080:8080 adviser-app
```

3. **브라우저에서 접속**
- `http://localhost:8080`

## 📖 사용 방법

### 1. 초기 진입
- 앱 실행 시 스플래시 화면 표시
- 평생 무료 서비스 안내 모달 표시
- 홈 화면으로 이동

### 2. 광고 분석 시작
1. 홈 화면에서 "내 가게 건강검진 지금 시작하기" 버튼 클릭
2. 면책 조항 확인 후 동의
3. 분석 화면으로 이동

### 3. 채널 선택
1. "어떤 광고를 분석할까요?" 화면에서 사용 중인 광고 플랫폼 선택
2. 각 플랫폼의 하위 옵션 중 해당하는 항목 선택
3. "다음" 버튼 클릭

### 4. 데이터 업로드
1. "데이터 입력" 화면에서 각 채널별로 관리자 페이지 스크린샷 업로드
2. 이미지 선택 또는 드래그 앤 드롭
3. OCR 자동 인식 대기
4. 인식된 데이터 확인 (노출수, 클릭수, 전환수)
5. 모든 채널 업로드 완료 후 "분석 시작하기" 버튼 클릭

### 5. 분석 결과 확인
1. 1단계: 분석 결과 확인
2. 2단계: 진단 결과 및 ROAS 차트 확인
3. 3단계: 개선 방안 확인

### 6. 리뷰 관리
1. 홈 화면에서 "리뷰" 탭 클릭
2. 답글 대기 중인 리뷰 확인
3. "AI 답글 생성하기" 버튼 클릭
4. 생성된 답글 중 선택하여 복사

### 7. 미션 완료
1. 홈 화면의 "오늘의 퀘스트"에서 미션 확인
2. 미션 클릭하여 완료
3. XP 획득 및 보상 모달 확인

## 🐳 Docker 지원

프로젝트는 Docker를 통해 컨테이너화되어 배포할 수 있습니다.

### Dockerfile 구조
```dockerfile
FROM node:20-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci --omit=dev
COPY . .
EXPOSE 8080
CMD ["npm", "start"]
```

### Docker 명령어
- 빌드: `docker build -t adviser-app .`
- 실행: `docker run -p 8080:8080 adviser-app`
- 백그라운드 실행: `docker run -d -p 8080:8080 adviser-app`
